Home Automation using IOT

Introduction :

This project has integration of Android mobile technology and embedded system. Android mobile user has to install an application on his mobile handset to control the devices. Then he/she can give command using the buttons on that application. For this you have to turn on the Bluetooth on mobile, so the main wireless controlling technique used in this project is Bluetooth technology. Bluetooth receiver will be connected to the project. This Bluetooth device is connected to the circuit which has a decoder. It sends out a code for respective command sent by user. Then the respective device connected to the circuit will be turned on or off depending on the command given. For example: Turn on motor, Turn off motor. Turn on buzzer etc. Such that by giving commands from mobile you can control home appliances.

Working Explanation :

  1. First, all the smart devices such as smart thermostats, smart plugs, smart lights, and smart security camera are installed in different rooms of the house.
  2. Next, these devices are connected to a central hub that acts as a control center for all the devices. The hub is usually Wi-Fi or Bluetooth enabled.
  3. The hub is then connected to the internet, enabling the user to control the devices remotely from a mobile app or website.
  4. The user can set up different rules and schedules for the devices to automate the routine tasks. For example, setting up a rule that turns off the lights and adjusts the thermostat when no one is in the room.
  5. The smart devices can also be controlled through voice commands using virtual assistants like Google Assistant or Amazon Alexa.

ADVANTAGES:

  1. Can control device from a Any where, thus it gives ease of access.
  2. Faster operation and efficient.
  3. No need to carry separate remote or any other controlling unit.
  4. Easy to operate and monitor the status of the devices.

APPLICATIONS:

  1. This project can be used to control various devices in the Home.
  2. Can also be used for security purpose after modification (we can control gate system or we can interface wireless camera and can control it using our mobile).
  3. It can be also used for controlling all devices in the home.
